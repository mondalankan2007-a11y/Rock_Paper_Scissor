# Rock_Paper_Scissor
# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
The player competes against the computer, which randomly selects its move.

## 🎯 How to Play

1. Click **Rock**, **Paper**, or **Scissor**.
2. The computer will randomly choose its move.
3. The winner is decided using the rules:

* Rock 🪨 beats Scissor ✂️
* Scissor ✂️ beats Paper 📄
* Paper 📄 beats Rock 🪨

4. The score updates after each round.
